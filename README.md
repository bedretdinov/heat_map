# heat_map
Тепловая карта для jupyter  

```python
from simplemap import showMap 
showMap(coordinates=[
    [55.748228, 47.660954],
    [55.748228, 47.650954],
    [55.748228, 47.680954],
    [55.748228, 47.790954],
], zoom=4,center='55.748228, 47.660954')
```
